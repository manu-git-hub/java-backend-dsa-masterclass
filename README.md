## 🛠️ The Backend Stack & DSA Focus
We focus on writing **Production-Grade Java**, prioritizing:
* **Clean Code:** Strict adherence to **SOLID principles** and **Advanced Design Patterns**.
* **Efficiency:** Strict adherence to $O(n)$ time and $O(1)$ space where possible.
* **Concurrency:** Deep dives into **Virtual Threads (Project Loom)** and synchronization.
* **Robustness:** Unit testing with JUnit 5 and automated edge-case validation.

## 🗺️ Curriculum Roadmap

### 📦 Phase 1: The Data Structure Core
* **Linear:** Dynamic Arrays, Advanced Linked Lists, Skip Lists.
* **Non-Linear:** Trees (Segment Trees, Fenwick Trees), Graphs (Shortest Path, MST).
* **High Performance:** ConcurrentHashMaps, Bloom Filters, and Tries.

### ⚙️ Phase 2: Algorithmic Patterns
* **Optimization:** Greedy, Dynamic Programming (Bottom-Up & Top-Down).
* **Traversals:** BFS/DFS, Backtracking, and Advanced Recursion.
* **Math:** Bit Manipulation, Number Theory for Cryptography.

### 🏗️ Phase 3: Backend & System Design (LLD)
* **Design Patterns:** Singleton, Factory, Strategy, and Observer patterns.
* **Concurrency:** Producer-Consumer, Thread Pools, and Deadlock Prevention using Virtual Threads.
* **Scalability:** Implementation of Caching (LRU/LFU) and Distributed Rate Limiters.
